# genicns
genicns is a simple command-line tool that generate ICNS file.

## Dependencies

### Necessary
* Bash

### Optional
* ExifTool (To strip metadata)

## Installation
```sh
git clone https://codeberg.org/qdz13/genicns
cd genicns
sudo make install
```
> [!NOTE]
> This program is only works on macOS.

## Basic Usage
```sh
genicns input.png -o output.icns
```
